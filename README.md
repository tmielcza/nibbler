# How to
```
make
./nibbler <options>
```

Eat the rings to earn points. The snake grows up the more it eats. Plenty (5) of cool bonuses. Don't eat your tail nor a wall.

Arrows / WASD to move.

Local / Network multiplayer.

Optionnal edges.
